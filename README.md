# fetch
[fetch](https://github.com/r2dr0dn/fetch) Is A Command Line Utility To Fetch Websites Source Code.


```
            Installation
Ubuntu/Debian
sudo apt-get golang
Arch
sudo pacman -S go
```

```
            Running
go run fetch.go            
```

```
           Building
go build fetch.go
           Running
./fetch          
```
